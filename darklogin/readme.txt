This applicication is a simple python Django login interface.

this application includes the following

- Captcha Secuirty 
- Email and username input feilds
- input feild validation
- currently not setup for production USE current in DEBUG MODE
- login and register interface 
- redirect after login
- redirect after register & auto login
- credential saving after page refresh 
- supports mobile and desktop browsers
- supports all modern browsers(Chrome, Brave, Edge, Safari, ETC)
